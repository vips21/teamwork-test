# Teamwork

#### Pre-requisites
    Python 3.7

#### Installation
    Clone repo
    Create environment with Python 3.7
    python3.7 -m virtualenv MyEnv
    source MyEnv/bin/activate
    pip install -r requirements.txt

### Create super user
    cd teamwork
    python manage.py createsuperuser

### Run server
    python manage.py runserver
